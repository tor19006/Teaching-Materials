01 ส่วนที่ 1

จังหวะ คืออะไร?

ทำความรู้จัก Beat, Rhythm, Tempo และ BPM
ไม่ต้องอ่านโน้ต — แค่ฟังและนับตามก็เข้าใจได้!

แนวคิดพื้นฐาน

🎵 จังหวะ (Rhythm) คืออะไร?

จังหวะ = รูปแบบของเสียงและความเงียบที่เรียงตัวกันตามเวลา — เป็นสิ่งที่ทำให้ดนตรีน่าสนใจ ไม่ใช่แค่เสียง "ติ๊กๆ" เท่ากัน

💓 จังหวะอยู่รอบตัวเรา

  • • เสียงนาฬิกา ติ๊ก-ต๊อก-ติ๊ก-ต๊อก
  • • จังหวะการเดิน ก้าว-ก้าว-ก้าว-ก้าว
  • • เสียงฝนตกหยด ป๊ะ..ป๊ะ..ป๊ะ..
  • • เสียงพิมพ์แชท ต๊ะ-ต๊ะ-ต๊ะต๊ะ-ต๊ะ
  • • หัวใจเต้น ตุ๊บ-ตุ๊บ-ตุ๊บ-ตุ๊บ

🧠 ทุกคนมีจังหวะในตัว!

ลองเอามือวางบนหน้าอก แล้วรู้สึกการเต้นของหัวใจ — นั่นคือจังหวะ!

เวลาเราพูด บางคำสั้น บางคำยาว — นั่นก็คือจังหวะเช่นกัน!

🔑

สูตรง่ายๆ: จังหวะ = เสียง + ความเงียบ + เวลา

เปรียบเทียบ

🥊 Beat กับ Rhythm ต่างกันยังไง?

คือชีพจรสม่ำเสมอ เหมือนหัวใจลวดลายของเสียง เหมือนท่าเต้น
ลักษณะคงที่ ไม่เปลี่ยน
ติ๊ก-ติ๊ก-ติ๊ก-ติ๊ก		เปลี่ยนไปเรื่อย
ตุ๊ม-แตก-ตุ๊ม ตุ๊ม-แตก
เปรียบ🚶 การเดิน — ก้าวสม่ำเสมอ💃 การเต้นรำ — หลากหลายท่า
ลองทำเคาะเท้าตามเพลงปรบมือตามทำนองร้อง
🎧

ลองทำตาม: เปิดเพลงที่ชอบ → เคาะเท้าตามจังหวะ (= Beat) → ปรบมือตามทำนองร้อง (= Rhythm) → สังเกต: เท้าเคาะสม่ำเสมอ แต่มือปรบแตกต่างกันในแต่ละท่อน!

ความเร็วจังหวะ

⏱️ Tempo & BPM — เพลงเร็วเพลงช้าวัดยังไง?

Tempo (เทมโป) = ความเร็วของ Beat — วัดเป็น BPM (Beats Per Minute) หรือจำนวนจังหวะต่อนาที

แนวเพลงBPMอารมณ์
🐢 บัลลาด/Slow60–80ช้า เศร้า ซึ้ง
🚶 Hip-Hop/R&B85–115ชิลๆ มีสไตล์
🏃 Pop100–130สนุก ฟังสบาย
🏃‍♂️ Rock/EDM120–140ตื่นเต้น พลังสูง
🚀 Drum & Bass160–180เร็วมาก ลุ้นระทึก

📏 ลองจับเวลาดู!

  • • 60 BPM = ตบมือ 1 ครั้งต่อวินาที (เท่าเข็มวินาที)
  • • 120 BPM = ตบมือ 2 ครั้งต่อวินาที (เท่าจังหวะเดินเร็ว)
  • • 180 BPM = ตบมือ 3 ครั้งต่อวินาที (เร็วมาก!)
💡

BPM สูง = เร็ว = ตื่นเต้น
BPM ต่ำ = ช้า = สงบ/เศร้า

รูปแบบจังหวะ

🔄 Pattern (แพทเทิร์น) คืออะไร?

Pattern = รูปแบบจังหวะที่ทำซ้ำได้ เหมือนลายผ้า — ทุกเพลงมี Pattern ตั้งแต่ K-Pop ไปจนถึงเพลงลูกทุ่ง!

📐 จังหวะ 4/4 (สี่สี่)

จังหวะที่ใช้บ่อยที่สุดในโลก! นับ 1-2-3-4 แล้วเริ่มใหม่

เพลงป๊อป ร็อค ฮิปฮอป แร็ป EDM ลูกทุ่ง — ส่วนใหญ่ใช้ 4/4 ทั้งนั้น!

🧩 วิธีสร้าง Pattern ง่ายๆ

  • • ขั้น 1: นับ 1-2-3-4 ให้สม่ำเสมอ
  • • ขั้น 2: เลือกเสียง 2-3 แบบ
  • • ขั้น 3: เลือกว่าจะ "ตี" จังหวะไหน "เงียบ" จังหวะไหน
  • • ขั้น 4: ทำซ้ำ! แค่นี้ก็ได้ Pattern แล้ว!

Object Percussion
เปลี่ยนสิ่งของเป็นเครื่องดนตรี

โต๊ะ ปากกา ขวดน้ำ หนังสือ ตบมือ ดีดนิ้ว กระทืบเท้า
ทุกอย่างรอบตัวสามารถเป็นเครื่องดนตรีได้!

สิ่งของรอบห้องเรียน

🪑 Object Percussion — เสียงจากสิ่งของ

ไม่ต้องมีกลองจริง — ใช้สิ่งของบนโต๊ะเรียนก็สร้าง Beat ได้!

สิ่งของวิธีทำเสียงเสียงคล้ายหน้าที่ใน Beat
🪑 โต๊ะตบฝ่ามือ / กำหมัดเคาะKick (กลองเบส)จังหวะหนัก 💥
🖊️ ปากกาเคาะกับโต๊ะ / กลิ้งเคาะHi-hat (ฉาบ)จังหวะถี่เบาๆ
📖 หนังสือปิดดังๆ / ตบ / เคาะSnare (กลองสแนร์)จังหวะเน้น 👊
🧴 ขวดน้ำเคาะด้านข้าง / เขย่าShaker / Effectเพิ่มสีสัน ✨
📏 ไม้บรรทัดเคาะกับโต๊ะ / สั่นเสียงWoodblock / Rimจังหวะเสริม
💡

เคล็ดลับ: เสียงจะเปลี่ยนตามตำแหน่งที่เคาะ! ลองเคาะ กลางโต๊ะ vs ขอบโต๊ะ, ขวดมีน้ำ vs ขวดเปล่า, ตบมือแบน vs ถ้วยมือ — เสียงต่างกันมาก!

สรุปเนื้อหาสำคัญและคาบเรียนต่อไป

วันนี้เราเปลี่ยนจาก "ฉากหลัง" (Ambience) มาสู่ "หัวใจ" (Rhythm & Beat) ของดนตรี!

💡 สิ่งที่เก็บกลับบ้านวันนี้

  • • Beat = ชีพจรสม่ำเสมอ · Rhythm = ลวดลายบน Beat
  • • Tempo/BPM = ความเร็วจังหวะ (ตัวเลขสูง = เร็ว)
  • • Pattern = จังหวะที่ทำซ้ำ สร้างจากการนับ 1-2-3-4
  • • Object Percussion = ทุกสิ่งรอบตัวเป็นเครื่องดนตรีได้!
  • • Kick, Snare, Hi-hat = บทบาทเสียงในจังหวะ

🧠 ควิซประเมินผลทบทวนความเข้าใจ (10 ข้อ)

1. "จังหวะ" (Rhythm) หมายถึงอะไร?
ก ความดังของเสียง
ข รูปแบบของเสียงและความเงียบที่เรียงตัวกันตามเวลา
ค ระดับเสียงสูงต่ำ
ง ชนิดของเครื่องดนตรี
2. Beat กับ Rhythm ต่างกันอย่างไร?
ก Beat คือชีพจรสม่ำเสมอ แต่ Rhythm คือลวดลายที่เปลี่ยนไป
ข Beat กับ Rhythm เป็นสิ่งเดียวกัน แค่เรียกต่างภาษา
ค Beat ใช้กับเพลงเร็ว ส่วน Rhythm ใช้กับเพลงช้า
ง Beat ใช้กลอง ส่วน Rhythm ใช้กีตาร์
3. BPM ย่อมาจากอะไร?
ก Basic Pitch Measurement
ข Band Production Method
ค Beats Per Minute (จำนวนจังหวะต่อนาที)
ง Bass Performance Mode
4. เพลง Pop ทั่วไปมี BPM ประมาณเท่าไหร่?
ก 30-50 BPM
ข 100-130 BPM
ค 180-220 BPM
ง 300-400 BPM
5. "Pattern" ในทางดนตรีหมายถึงอะไร?
ก เพลงที่มีเนื้อร้องยาว
ข รูปแบบจังหวะที่ทำซ้ำได้
ค เครื่องดนตรีราคาแพง
ง โน้ตเพลงที่ซับซ้อนมาก
6. ในการสร้าง Beat Pattern จากสิ่งของ เสียงเคาะโต๊ะ (เสียงทุ้มหนัก) ทำหน้าที่คล้ายเสียงใด?
ก Hi-hat (ฉาบ)
ข Kick (กลองเบส)
ค Flute (ขลุ่ย)
ง Violin (ไวโอลิน)
7. ถ้าต้องอัดเสียงเคาะโต๊ะด้วยมือถือ ควรทำอย่างไร?
ก วางมือถือบนโต๊ะที่เคาะ เสียงจะชัดที่สุด
ข ถือมือถือห่าง 30-60 ซม. ไม่วางบนโต๊ะที่เคาะ
ค วางมือถือไว้ในกระเป๋า จะไม่มีเสียงรบกวน
ง ถือชิดปากให้มากที่สุด
8. ใน Beat Pattern พื้นฐาน (4/4) เสียง Snare (เช่น ตบมือ) มักตีจังหวะใด?
ก จังหวะ 1 และ 3
ข จังหวะ 2 และ 4
ค ทุกจังหวะ 1-2-3-4
ง ไม่มีจังหวะตายตัว
9. "Body Percussion" คืออะไร?
ก การเล่นกลองชุดจริง
ข การใช้ร่างกายสร้างเสียงจังหวะ เช่น ตบมือ ตบขา กระทืบเท้า
ค การร้องเพลง
ง การออกกำลังกายประกอบจังหวะ
10. STOMP คืออะไร?
ก แอปพลิเคชันตัดต่อเสียง
ข กลุ่มศิลปินที่สร้างดนตรีจากสิ่งของในชีวิตประจำวัน
ค เทคนิคการร้องเพลงสมัยใหม่
ง เครื่องดนตรีไฟฟ้าแบบใหม่
