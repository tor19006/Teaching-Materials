🥊 Beat กับ Rhythm ต่างกันยังไง?

💓

Beat = ชีพจรสม่ำเสมอ

เหมือน เข็มวินาทีนาฬิกา — ติ๊ก ติ๊ก ติ๊ก ติ๊ก
หรือเหมือน ก้าวเท้าเดิน — ก้าว ก้าว ก้าว ก้าว

คงที่ สม่ำเสมอ ไม่เปลี่ยน

🎧 ลองทำ: เคาะเท้าตามเพลงที่ชอบ → นั่นคือ Beat!

💃

Rhythm = ลวดลายที่เปลี่ยนไป

เหมือน ท่าเต้น ที่ทำระหว่างก้าวเท้า — มีท่าสั้น ท่ายาว ท่าหยุด

เปลี่ยนแปลง หลากหลาย มีสั้นมียาว

🎧 ลองทำ: ปรบมือตามทำนองร้อง → นั่นคือ Rhythm!

🔑

จำง่ายๆ: Beat = การเดิน (สม่ำเสมอ) · Rhythm = การเต้นรำ (หลากหลาย) · ทั้งสองอย่างรวมกัน = ดนตรี!

พื้นฐาน DAW

🎛️ DAW คืออะไร? ทำไมต้องใช้?

DAW (Digital Audio Workstation) = สตูดิโอเพลงบนมือถือ/คอมพิวเตอร์

  • • อัดเสียง จากไมค์มือถือได้ทันที
  • • นำเข้าไฟล์เสียง ที่อัดไว้ก่อนหน้า (เสียง Ambience!)
  • • จัดวางหลาย Track ซ้อนกัน เล่นพร้อมกัน
  • • ปรับ Volume, Fade, ตัดต่อ ได้ง่ายๆ
  • • ส่งออกเป็นไฟล์เสียง พร้อมแชร์/ส่งงาน
💡

เปรียบเทียบ: ถ้า Voice Memos เหมือน "กล้องถ่ายรูป" → DAW เหมือน "Photoshop" สำหรับเสียง — ทำอะไรได้มากกว่าเยอะ!

คำศัพท์ DAW

📖 ศัพท์ที่ต้องรู้ (ใช้บ่อยมาก!)

แค่ 8 คำนี้ก็ทำงานใน BandLab ได้แล้ว

🛤️ Track (แทร็ค)

ช่องเสียง 1 ช่อง
เหมือน "ราง 1 ราง"
แต่ละเสียง = 1 Track

📦 Region (ก้อนเสียง)

ก้อนเสียงที่อยู่บน Track
มองเห็นเป็น คลื่นเสียง
ลากเลื่อนได้

📏 Timeline (เส้นเวลา)

แกนเวลา ซ้ายไปขวา
ซ้าย = เริ่ม, ขวา = จบ
ซูมเข้า-ออกได้

🎚️ Volume (วอลุ่ม)

ความดังของแต่ละ Track
Slider ซ้ายเบา ขวาดัง
ปรับแยกแต่ละ Track

🔇 Mute (มิวท์)

ปิดเสียง Track
กดปุ่ม M
เสียงยังอยู่ แค่เงียบ

🎤 Solo (โซโล)

ฟังแค่ Track เดียว
กดปุ่ม S
Track อื่นเงียบหมด

🌅 Fade (เฟด)

เสียงค่อยๆ ดัง/เบา
Fade In = ค่อยๆ ดัง
Fade Out = ค่อยๆ เบา

📤 Export (ส่งออก)

รวมทุก Track เป็น 1 ไฟล์
ได้ MP3 ส่งงานได้
เมนู Share → Mixdown

ขั้นตอนที่ 3 · ใช้เสียงจากคาบก่อน!

📂 Import เสียง Ambience เข้า BandLab

เอาเสียงบรรยากาศที่อัดไว้ตั้งแต่คาบที่ 2-3 มาวางเป็น Track แรก!

1

กดปุ่ม + (เพิ่ม Track) ในหน้า Mix Editor

อยู่ฝั่งซ้าย ใต้ Track ที่มีอยู่ (หรือตรงกลางถ้ายังว่าง)

2

เลือก "Import"

จะเปิดหน้า Files ของมือถือ

3

เลือกไฟล์เสียง Ambience ที่อัดไว้

iOS: ถ้าอัดด้วย Voice Memos → ต้อง Share → Save to Files ก่อน
Android: หาได้จากโฟลเดอร์ Recordings/Downloads

✓

เสียง Ambience ปรากฏเป็น Track แรก!

จะเห็นเป็นคลื่นเสียง (Waveform) บน Timeline — ลองกด ▶ ฟังดู!

✅ ไฟล์ที่รองรับ

  • • ✅ .m4a (Voice Memos)
  • • ✅ .mp3
  • • ✅ .wav
  • • ✅ .aac / .flac / .ogg
🔥

ไม่มีไฟล์ Ambience? ไม่เป็นไร! กดปุ่ม + → เลือก "Import" → เปิด Loops Library → ค้นหา "ambient" หรือ "nature" → เลือกมาใช้แทนได้!

👆

Tips: เปลี่ยนชื่อ Track โดยกดที่ชื่อ Track → พิมพ์ "Ambience" เพื่อไม่ให้สับสน

ขั้นตอนที่ 4 · อัด Track ที่ 2!

🎙️ อัดเสียง Object Percussion ลง BandLab

อัดเสียงเคาะสิ่งของ ตรงในแอป BandLab เลย! ไม่ต้องอัดแยกแล้วค่อย import

1

กดปุ่ม + → เลือก "Voice/Mic"

BandLab จะเพิ่ม Track ใหม่ + เปิดไมค์มือถือ
⚠️ ครั้งแรก: อนุญาตการเข้าถึงไมค์ (กด Allow!)

2

🎧 ใส่หูฟัง 1 ข้าง (สำคัญ!)

เพื่อฟัง Track Ambience ขณะอัดเสียง Beat
ถ้าไม่ใส่ = เสียง Ambience จะรั่วเข้าไมค์

3

กด ⏺ Record → นับ 3-2-1 → เคาะ!

เคาะตามจังหวะ 1-2-3-4 ที่ฝึกไว้
ทำซ้ำ Pattern ประมาณ 15-30 วินาที

4

กด ⏹ หยุด → ฟังผลลัพธ์

กด ▶ ฟังทั้ง 2 Track พร้อมกัน → ได้ยิน Ambience + Beat!
ไม่พอใจ? Undo แล้วอัดใหม่ได้ไม่จำกัด!

💡 เคล็ดลับอัดเสียงให้ดี

  • • ถือมือถือห่าง 30-50 ซม. จากวัตถุ
  • • เปิด Metronome ใน BandLab ช่วยนับจังหวะ
  • • อัดทดสอบสั้นๆ ก่อน 5 วินาที → ฟังดู
  • • อย่าวางมือถือบนโต๊ะที่เคาะ! จะสั่น
  • • หาที่เงียบ ปิดพัดลม ลดเสียงรบกวน
  • • เสียงแตก? → ถอยออก หรือเคาะเบาลง
🎵

Overdub: BandLab จะเล่น Track เดิม (Ambience) ให้ฟังทางหูฟังขณะอัด — เหมือนเล่นดนตรีตามเพื่อน!

ขั้นตอนที่ 5

🎚️ ปรับ Volume + Fade → Mix 2 Track เข้าด้วยกัน

ตอนนี้มี 2 Track แล้ว — ปรับให้เสียงกลมกล่อมเข้ากัน

🎚️

ปรับ Volume

แต่ละ Track มี Slider ปรับเสียง

Track Ambience: ลด Volume ลงนิด
(ให้เป็นพื้นหลัง ไม่ดังเท่า Beat)

Track Beat: ปรับให้พอดี ไม่ทับ Ambience

🌅

ทำ Fade

กดที่ Region (ก้อนเสียง) → ลากจุดมุมบน

Fade In = ลากมุมซ้ายบน → เสียงค่อยๆ ดังขึ้น

Fade Out = ลากมุมขวาบน → เสียงค่อยๆ เบาลง

ทำให้เสียงเริ่ม/จบอย่างนุ่มนวล ไม่กระตุก!

✂️

Trim (ตัดส่วนเกิน)

กดที่ Region → ลากขอบซ้าย/ขวาเข้ามา

ตัดส่วนเงียบๆ ตอนเริ่มออก

ตัดส่วนท้ายที่ไม่ต้องการออก

ง่ายมาก แค่ลากขอบ!

🎧

ลองทำ: กดปุ่ม S (Solo) ที่ Track Ambience → ฟังแค่ Ambience · กด S ที่ Track Beat → ฟังแค่ Beat · ปิด Solo → ฟังรวม → ปรับ Volume จนพอใจ!

ขั้นตอนที่ 6 · ส่งออกผลงาน!

📤 Export — รวม Track เป็นไฟล์เสียง

เสร็จแล้ว! ถึงเวลาส่งออกผลงานเป็น MP3 สำหรับส่งงาน

1

กดปุ่ม Share/⋮ (มุมขวาบน)

จะมีเมนูตัวเลือกขึ้นมา

2

เลือก "Mixdown" หรือ "Download Mix"

BandLab จะรวมทุก Track เป็นไฟล์เดียว

3

เลือกรูปแบบ: MP3 (แนะนำ)

ไฟล์ขนาดเล็ก ส่งง่าย ใช้ได้ทุกที่

4

ส่งงาน: อัพโหลดไฟล์ MP3 ลง Google Classroom

หรือ: กด Share → Publish → Copy Link → วางลงใน Classroom

🎯 Checklist ก่อนส่ง

  • ✅ มี Track Ambience เป็นพื้นหลัง
  • ✅ มี Track Beat/Percussion เป็นจังหวะ
  • ✅ ปรับ Volume ให้เสียงเข้ากัน
  • ✅ ทำ Fade In/Out อย่างน้อย 1 จุด
  • ✅ ความยาว 15-30 วินาที
  • ✅ ตั้งชื่อ Project ชัดเจน
  • ✅ Export เป็น MP3 แล้ว

📝 สรุปสิ่งที่ได้เรียนรู้

คาบนี้เราเรียนทั้งทฤษฎีจังหวะ + ลงมือใช้ DAW จริง!

🥁 จังหวะพื้นฐาน

  • • Beat = ชีพจรสม่ำเสมอ (เหมือนเดิน)
  • • Rhythm = ลวดลายที่ซ้อนบน Beat (เหมือนเต้น)
  • • Pattern = จังหวะที่ทำซ้ำ (นับ 1-2-3-4)
  • • Object Percussion = เสียงจากสิ่งของรอบตัว

🎛️ BandLab (DAW)

  • • DAW = สตูดิโอเพลงบนมือถือ
  • • สร้าง Project → Import เสียง → อัด เสียงใหม่
  • • ปรับ Volume, ทำ Fade, Trim
  • • Export เป็น MP3 ส่งงาน
  • • Mute/Solo ฟังแยก Track

🧠 ควิซทบทวนความเข้าใจ (10 ข้อ)

1. DAW ย่อมาจากอะไร?
ก Digital Audio Workstation
ข Digital Audio Website
ค Data And Wireless
ง Download Any Wave
2. "Track" ใน DAW หมายถึงอะไร?
ก ช่องเสียง 1 ช่อง ใช้วางเสียง 1 ประเภท
ข เพลงที่ดาวน์โหลดจากอินเทอร์เน็ต
ค ลำโพงของโทรศัพท์มือถือ
ง ปุ่มอัดเสียงของแอป
3. ถ้าต้องการนำไฟล์เสียงที่อัดไว้เข้า BandLab ต้องทำอย่างไร?
ก กดปุ่ม + → เลือก Import
ข ถ่ายภาพหน้าจอ Voice Memos
ค กดปุ่ม Share → Facebook
ง ส่งไฟล์ผ่าน Line ให้ครู
4. ปุ่ม "Mute" (M) บน Track ทำหน้าที่อะไร?
ก ลบ Track ออกถาวร
ข ปิดเสียง Track นั้น (เสียงยังอยู่ แค่เงียบ)
ค เพิ่ม Volume ให้ดังขึ้น
ง ส่งเสียงไปให้เพื่อนในกลุ่ม
5. "Fade Out" หมายถึงอะไร?
ก เสียงหายไปทันที
ข เสียงค่อยๆ เบาลงจนเงียบ
ค เสียงค่อยๆ ดังขึ้น
ง เสียงเปลี่ยนจากซ้ายไปขวา
6. Beat กับ Rhythm ต่างกันอย่างไร?
ก Beat = ชีพจรสม่ำเสมอ, Rhythm = ลวดลายที่เปลี่ยนไป
ข Beat ใช้กับเพลงเร็ว, Rhythm ใช้กับเพลงช้า
ค Beat กับ Rhythm เป็นสิ่งเดียวกัน
ง Beat ใช้กลอง ส่วน Rhythm ใช้กีตาร์
7. ถ้าต้องการอัดเสียงตรงใน BandLab ต้องกดปุ่มอะไร?
ก กดปุ่ม + → Voice/Mic → แล้วกด ⏺ Record
ข กด Share → Mixdown
ค กดปุ่ม M บน Track
ง กดปุ่ม ▶ Play
8. ทำไมต้องใส่หูฟังตอนอัดเสียง Percussion ทับ Track Ambience?
ก เพื่อให้เสียงดังขึ้น
ข เพื่อไม่ให้เสียง Ambience จากลำโพงรั่วเข้าไมค์
ค เพราะ BandLab บังคับให้ใส่
ง เพื่อให้ดูเท่
9. "Export" หรือ "Mixdown" คืออะไร?
ก ลบ Project ทิ้ง
ข รวมทุก Track เป็นไฟล์เสียง 1 ไฟล์ (เช่น MP3)
ค เพิ่ม Track ใหม่
ง เปลี่ยนชื่อ Project
10. ถ้าเสียงที่อัดแตก (distort) ควรทำยังไง?
ก เพิ่ม Volume ให้ดังขึ้น
ข ถือมือถือห่างขึ้น หรือเคาะเบาลง
ค วางมือถือบนโต๊ะที่เคาะ
ง ลบแอปแล้วโหลดใหม่
